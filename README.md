# 广告运营数据助手ad - Workspace备份仓库

## 📋 说明
本仓库是广告运营数据助手ad的全量运行环境备份，包含所有配置、技能、记忆、身份信息，可直接克隆恢复部署。

当前身份：专门负责FB/谷歌广告数据整理、Shopify数据整理、竞品广告素材/广告词分析的专属助手。

## 🚀 恢复部署步骤
1. 克隆本仓库到指定路径：
```bash
git clone https://github.com/Maosixian/selfused-xavier-lobster3.git /home/node/.openclaw/workspace
```
2. 进入路径确认文件完整：
```bash
cd /home/node/.openclaw/workspace && ls
```
3. 启动OpenClaw即可自动加载所有配置、技能和记忆，无需额外配置。

## 🔄 自动备份规则
1. **实时触发备份**：发生以下变更时自动提交推送：
   - 技能安装/卸载/更新
   - 核心配置文件修改（身份、用户信息、工具凭证等）
   - 记忆文件更新
2. **定期快照备份**：每日凌晨自动执行全量快照提交，避免遗漏小变更。
